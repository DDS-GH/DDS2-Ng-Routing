Notes

Shell Inheritance

All wrapping components inherit from lib/helpers/dds.component.ts

Initialization Timing

The component shell normally initializes the DDS component within the AfterViewInit life cycle stage. However if you add the init property to any of the components' instances, it will initialize AFTER that step, whenever the DDS component is found in the DOM.
